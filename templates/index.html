ZJountries-api

About

This project is made in python inspired by restcountries.com.

API version: {{ api_version }}

All:

You can call everything by using all

https://{{ domain }}/{{ api_ver }}/all

/{{ api_ver }}/all

Name:

Call any country by their name

https://{{ domain }}/{{ api_ver }}/name={name}

/{{ api_ver }}/name=poland

Fulltext

Call a country by their full name.

https://{{ domain }}/{{ api_ver }}/name={name}?fulltext=true

/{{ api_ver }}/name=poland?fulltext=true

Continent:

Call countries that are in a continent

https://{{ domain }}/{{ api_ver }}/continent={continent}

/{{ api_ver }}/continent=europe

ISO codes:

Can call any place that has an ISO-3166 code

https://{{ domain }}/{{ api_ver }}/iso2={alpha2}

/{{ api_ver }}/iso2=pl

https://{{ domain }}/{{ api_ver }}/iso3={alpha3}

/{{ api_ver }}/iso3=pol

https://{{ domain }}/{{ api_ver }}/isoN={numeric}

/{{ api_ver }}/isoN=616

Language:

Call countries that speak whatever language.

https://{{ domain }}/{{ api_ver }}/language={lang}

/{{ api_ver }}/language=polish

Official

Calls those who's that language is official.

https://{{ domain }}/{{ api_ver }}/language={lang}?official=true

/{{ api_ver }}/language=polish?official=true

Spoken

Calls those who's that language is spoken casually.

https://{{ domain }}/{{ api_ver }}/language={lang}?spoken=true

/{{ api_ver }}/language=polish?spoken=true

Fields

Have which keys should be visible when calling something by its field.

https://{{ domain }}/{{ api_ver }}/all?fields=name

/{{ api_ver }}/all?fields=name,capital

https://{{ domain }}/{{ api_ver }}/name=poland?fields=name,continent,population

/{{ api_ver }}/name=poland?fields=name,continent,population

Multiple

Can call multiple places at once, separate with: ,

https://{{ domain }}/{{ api_ver }}/iso2={alpha2},{alpha2},{alpha2}

/{{ api_ver }}/iso2=pl,ca,de
