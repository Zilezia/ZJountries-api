ZJountries-api

About

This project is made in python inspired by restcountries.com.

API version: {{ api_version }}

All:

You can call everything by using all

https://{{ domain }}/{{ api_ver }}/all

/{{ api_ver }}/all

Name:

Call any country by their name

https://{{ domain }}/{{ api_ver }}/name={name}

/{{ api_ver }}/name=poland

Fulltext

Call a country by their full name.

https://{{ domain }}/{{ api_ver }}/name={name}?fulltext=true

/{{ api_ver }}/name=poland?fulltext=true

ISO codes:

Can call any place that has an ISO-3166 code

https://{{ domain }}/{{ api_ver }}/iso2={alpha2}

/{{ api_ver }}/iso2=pl

https://{{ domain }}/{{ api_ver }}/iso3={alpha3}

/{{ api_ver }}/iso3=pol

https://{{ domain }}/{{ api_ver }}/isoN={numeric}

/{{ api_ver }}/isoN=616

Multiple

Can call multiple places at once, seperation is: ,

https://{{ domain }}/{{ api_ver }}/iso2={alpha2},{alpha2},{alpha2}

/{{ api_ver }}/iso2=pl,ca,de
